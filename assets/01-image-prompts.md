## Generation Tips

**For best results:**
- Add `--ar 16:9` for landscape scenes
- Add `--ar 3:4` or `--ar 4:5` for portrait/cover
- Use `--style raw` for more literal interpretation
- Add `children's book illustration, safe for kids, no scary elements` to all prompts
- Consider adding artist style references like "in the style of Oliver Jeffers" or "Eric Carle inspired"

**Consistency tips:**
- Generate all images in the same session if possible
- Use the same style modifiers across all prompts
- Create character reference sheets first, then use image-to-image consistency features

**Color palette:**
- Greens: vibrant, lush (jungle)
- Orange: warm sky
- Blues: Max's hoodie, hatch glow
- Purple/Pink: berries, next tunnel
- Earth tones: dinosaurs (browns, tans, greens)
